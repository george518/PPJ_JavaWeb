PPJ_JavaWeb
====
啥东西？What?
----
PPJ_JavaWeb是一个基于SpringMVC+Mybatis+Maven的web应用，内含RBAC权限管理模块。
可以直接用来开发相应的web应用后台。   
感觉不错，求star :)
